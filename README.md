# Netflix Customer Churn Prediction

## Objective
Predict whether a user will churn based on engagement and subscription data.

## Models Used
- Logistic Regression
- Random Forest

## Key Features
- Watch hours
- Last login days
- Subscription type

## Results
- Achieved ~80% accuracy
- Random Forest performed better

## Insights
- Low engagement users are more likely to churn
- Inactive users show high churn probability

## Business Impact
- Helps identify at-risk users
- Enables targeted retention strategies
